##verify on postman by using the respective routes and methods to perform CRUD operations
 on the items array::
1. Start the server by running the command `node app.js` in the terminal.
2. Use Postman to test the API endpoints:
   - To create a new item, send a POST request to `http://localhost:3000/items` with a JSON body containing the item details.
   - To retrieve all items, send a GET request to `http://localhost:3000/items`.
   - To update an item completely, send a PUT request to `http://localhost:3000/items/:id` with the updated item details in the body.
   - To update an item partially, send a PATCH request to `http://localhost:3000/items/:id` with the fields you want to update in the body.
   - To delete an item, send a DELETE request to `http://localhost:3000/items/:id`.
